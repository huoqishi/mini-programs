## 小程序的优势?

| 应用类型 | 性能 | 所需的运行环境 | 推广成本 |
| -------- | ---- | -------------- | -------- |
| 小程序   | 高   | 微信           | 低       |
| 网页     | 低   | 浏览器         | 低       |
| 手机软件 | 高   | 手机           | 高       |
